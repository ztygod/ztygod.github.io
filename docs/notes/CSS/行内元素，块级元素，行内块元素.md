## 块级元素（block-level）

+ <font style="color:rgb(37, 41, 51);">独占一行，其</font>**<font style="color:#DF2A3F;background-color:#FBDE28;">宽度自动填满其父元素宽度</font>**<font style="color:rgb(37, 41, 51);">（即使设置了宽度仍然是独占一行的）</font>
+ <font style="color:rgb(37, 41, 51);">多个块状元素标签写在一起，默认</font>**<font style="color:#DF2A3F;background-color:#FBDE28;">排列方式为从上至下</font>**
+ <font style="color:rgb(37, 41, 51);">高度，行高，外边距（margin）以及内边距（padding）都可以控制</font>

举例：<font style="color:rgb(255, 80, 44);background-color:rgb(255, 245, 245);">div ul ol li dl dt dd h1~h6 p</font>

## 行内元素（inline-level）

+ **<font style="color:#DF2A3F;background-color:#FBDE28;">不会独占一行</font>**，相邻的行内元素会排列在同一行里，直到一行排不下才会自动换行，其宽度随元素的内容而变化
+ 设置行高有效，等同于给父级元素设置行高
+ 高宽无效，**<font style="color:#DF2A3F;background-color:#FBDE28;">对外边距（margin）和内边距（padding）</font>**仅设置左右方向有效，上下无效。
+ 内边距（padding）设置上下从显示效果是增加的，但其实设置时无效的，并不会影响周围的元素
+ **<font style="color:#DF2A3F;background-color:#FBDE28;">行内元素中不能放块级元素</font>**，a 链接里面不能再放链接

举例：<font style="color:rgb(255, 80, 44);background-color:rgb(255, 245, 245);">a b span img input select strong</font>

## 行内块级元素（inline-block1）

+ <font style="color:rgb(37, 41, 51);">高度、行高、外边距以及内边距都可以控制；</font>
+ **<font style="color:#DF2A3F;background-color:#FBDE28;">默认宽度就是它本身内容的宽度，不独占一行</font>**<font style="color:rgb(37, 41, 51);">，但是之间会有空白缝隙，设置它上一级的 font-size 为 0，才会消除间隙；</font>

举例：==<input><img><select>==

## 元素类型转换

<font style="color:rgb(37, 41, 51);">display：block ，定义元素为块级元素</font>

<font style="color:rgb(37, 41, 51);">display : inline ，定义元素为行内元素</font>

<font style="color:rgb(37, 41, 51);">display：inline-block，定义元素为行内块级元素</font>
