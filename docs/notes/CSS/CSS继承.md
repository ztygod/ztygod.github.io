css中属性分为可继承属性与不可继承属性。

## 可继承属性

1. **<font style="background-color:#FBDE28;">字体系列属性</font>**

+ font-family：字体系列
+ font-weight：字体的粗细
+ font-size：字体的大小
+ font-style：字体的风格

2. **<font style="background-color:#FBDE28;">文本系列属性</font>**

+ text-indent：文本缩进
+ text-align：文本水平对齐
+ line-height：行高
+ word-spacing：单词之间的间距
+ letter-spacing：中文或者字母之间的间距
+ text-transform：控制文本大小写（就是uppercase、lowercase、capitalize这三个）
+ color：文本颜色

3. **<font style="background-color:#FBDE28;">元素可见性</font>**

+ visibility：控制元素显示隐藏

4. **列表布局属性**

+ list-style：列表风格，包括list-style-type、list-style-image等

## 不可继承属性

1. **<font style="background-color:#FBDE28;">display</font>**：规定元素应该生成的框的类型
2. **<font style="background-color:#FBDE28;">文本属性</font>**：

+ vertical-align：垂直文本对齐
+ text-decoration：规定添加到文本的装饰
+ text-shadow：文本阴影效果
+ white-space：空白符的处理
+ unicode-bidi：设置文本的方向

1. **<font style="background-color:#FBDE28;">盒子模型的属性</font>**：width、height、margin、border、padding
2. **<font style="background-color:#FBDE28;">背景属性</font>**：background、background-color、background-image、background-repeat、background-position、background-attachment
3. **<font style="background-color:#FBDE28;">定位属性</font>**<font style="background-color:#FBDE28;">：</font>float、clear、position、top、right、bottom、left、min-width、min-height、max-width、max-height、overflow、clip、z-index
4. **生成内容属性**：content、counter-reset、counter-increment
5. **轮廓样式属性**：outline-style、outline-width、outline-color、outline
6. **页面样式属性**：size、page-break-before、page-break-after
7. **声音样式属性**：pause-before、pause-after、pause、cue-before、cue-after、cue、play-during
