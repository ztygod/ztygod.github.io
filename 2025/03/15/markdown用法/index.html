Markdown 语法示例

Markdown 是一种轻量级标记语言，适用于写作和排版，广泛用于博客、文档和 GitHub README 文件等。本文将介绍 Markdown 的基本语法。

1. 标题

使用 # 号定义标题，# 代表一级标题，## 代表二级标题，以此类推。

2. 加粗 & 斜体

  • 加粗 使用 **加粗内容** 或 __加粗内容__
  • 斜体 使用 *斜体内容* 或 _斜体内容_
  • 加粗+斜体 使用 ***文本***

3. 列表

无序列表

使用 -、* 或 + 号创建无序列表。

有序列表

使用数字加 . 创建有序列表。

4. 代码块

使用反引号（`）表示代码。

行内代码

使用单个反引号包裹： console.log('Hello, Markdown!');

代码块

使用三个反引号包裹代码块：

图片

6. 引用 & 分割线

引用

使用 > 符号添加引用。

分割线

使用 --- 或 *** 创建分割线。

7. 表格

使用 | 分隔表格内容，- 定义表头。

8. 任务列表

使用 - [ ] 创建任务列表，- [x] 代表已完成的任务。

结语

Markdown 语法简洁高效，适用于博客、文档和笔记，希望本文能帮助你快速上手 Markdown！
